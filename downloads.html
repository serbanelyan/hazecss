HAZE

The class-less stylesheet

Instalation's pretty simple, just include the link in your website's header.
Haze Base only includes a class-less CSS file, available in two variants.

Mimified version:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze.min.css">

Non-mimified version: 

(show)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze.css">

HAZE +

The helpers and the utility

Same process as before, just include the link in your website's header.
Haze+ only includes a CSS file, available in two variants.

Mimified version:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze-plus.min.css">

Non-mimified version: 

(show)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze-plus.css">

HAZE ++

The UI components and scripts

Just a tad more complicatedjust include the css in your website's header and the script in the footer.
Haze++ required both a CSS and a JS file; Each of them is available in two variants.

Mimified version:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze-plus-plus.min.css">
<script src="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/js/haze.min.js" defer></script>

Non-mimified version: 

(show)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/css/haze/haze-plus-plus.css">
<script src="https://cdn.jsdelivr.net/gh/serbanelyan/hazecss@1.0.0/src/js/haze.js" defer></script>
